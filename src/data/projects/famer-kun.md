# FAMER-KUN Store

**Role:** Web Designer / Developer  
**Timeline:** 05/2022 - 07/2022  
**Status:** Completed (Graduation Project)  
**Team Size:** 1

## 🌟 Overview

FAMER-KUN Store is an intuitive e-commerce website built specifically as my final graduation project. Developed entirely on the **Wix platform**, this project showcases the power of no-code/low-code development. By utilizing Wix's drag-and-drop visual builder, I was able to rapidly construct a fully functional online agricultural storefront with a strong focus on aesthetics and user experience without writing complex traditional code.

## 🚀 Key Features

- **Visual E-Commerce Design:** Complete end-to-end setup of the shopping experience, including product galleries, shopping cart, and checkout workflows entirely through Wix's built-in platform features.
- **Agricultural-Themed UI:** Tailored, responsive layouts engineered to reflect a fresh, natural aesthetic suitable for an agricultural product store.
- **Rapid Delivery:** Demonstrated the ability to deploy a market-ready, visually stunning web application within a strict academic timeframe.

## 🛠️ Technical Stack

- **Platform:** Wix (No-code Website Builder)
- **Methodology:** Drag-and-drop Visual Editing, Responsive Design, UX/UI Optimization

## 💡 Project Highlights

- Successfully conceptualized, designed, and implemented a robust e-commerce solution within a tight **2-month graduation project timeframe**.
- Rather than focusing on underlying code architecture, this project drastically honed my skills in **User Interface (UI) design**, **User Experience (UX)**, and customer journey mapping.
- Proved versatility and adaptability by mastering a low-code platform to achieve concrete business & academic goals efficiently.
